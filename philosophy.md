# Notes on the philosophy of the project.

- Open Source.
- Data present here is representative, not comprehensive. Some subtility will be lost and exhaustivity is not an objective.
- Although this project tries to be as truthful and accurate as possible, we're not science specialists and we cannot be held
accountable for the validity of it.
- Open to multiple human languages.
- Contributions are welcome.
- Main purpose is educationnal.


## Possible points

- Easy to embed into any application, as all data is open source as well as typed.
- Data structuration is thought for extensibility first in case many more data types are added.
- Application was first intended for use in a Server Side Rendering environment, doing Client Side Rendering will require some
adaptation work.
